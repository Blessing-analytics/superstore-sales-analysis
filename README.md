# Superstore Sales Analysis

## Project Overview
This project analyzes a retail superstore dataset to uncover insights related to sales performance, profitability, customer segments, regional performance, shipping modes, and discount impact. The goal is to provide data-driven insights to support better business decisions.

## Dataset
- Source: Superstore retail dataset
- Records include sales, profit, category, sub-category, customer segment, region, discount, and shipping information.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Technology category generated the highest profit, while Furniture generated the lowest.
- Some sub-categories such as Tables, Bookcases, and Supplies consistently recorded losses.
- Higher discounts are associated with lower profitability.
- The Consumer segment contributed the most profit.
- Standard Class shipping generated higher profit than expedited shipping options.
